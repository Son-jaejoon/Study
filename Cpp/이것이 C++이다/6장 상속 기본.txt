1. 상속 : 객체 단위 코드를 재사용하는 방법, 재사용이란 기능적 확장이나 개선을 의미한다.
- 자식 클래스의 경우 먼저 호출되더라도 부모 클래스의 호출 이후
   부모 클래스 생성자 실행 및 반환 -> 자식 클래스 생성자 실행 및 반환이 된다.
   (생성자는 호출과 실행 순서 역순 / 소멸자는 동일)

2. 재정의 : 기존의 선언 및 정의된 코드를 유지하면서도 새롭게 바꾸는 방법. 즉, 코드를 자유롭게 대체 하는 것
- public : using 부모클래스::부모클래스; (C++11부터 생성자 상속을 지원함)

3. 메서드 재정의 : 어떤 클래스에 있는 메서드를 자유롭게 재정의해서 사용하는 방법.
